  • PerfOps

    Основатель стартапа

    • Закрыл 2 раунда инвестиций с 2мя Венчурными Фондами из Великобритании и Нидерландов
    • B2B SaaS платформа для оптимизации производительности глобальных интернет-сервисов
    • Наиболее известен благодаря сайтам DNSPerf и CDNPerf, которые предоставляют беспллатную и публичную информацию о производительности работы всех DNS и CDN провайдеров
    • Система аналитики собирает миллиарды RUM тестов и миллионы синтетических тестов для понимания точной производительности и аптайма распределенных интернет-инфраструктур
    • Сетевые тесты по запросу из 260+ глобальных локаций
    • Умный глобальный балансировщик нагрузки, который может использовать данные из системы аналитики либо любые другие внешние данные для принятия решений в реальном времени для каждого индивидуального запроса. Например, для автоматизации Multi-CDN и Multi-Cloud систем
    • Балансировщик нагрузки позволяет написать собственную логику, используя javascript, для полного контроля маршрутизации траффика
    • Backend использует anycast маршрутизацию в множественных глобальных локациях
